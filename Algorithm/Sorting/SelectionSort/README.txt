选择排序

1、介绍
	每一次从待排序的数据元素中找出最小（最大）的一个元素，移动至序列的起始位置，直到所有的数据元素排序完成

2、分析
	i代表当前已经完成排序的序号，数组共有n个元素
	1）从剩下的[i+1,n-1]个数组元素中找到最大（最小）的数字num[j]
	2）将num[j]与数组起始位置的元素进行替换

	=》
	1）相同元素的排列顺序在排序后会发生改变，故选择排序是不稳定的

3、时间复杂度
	第一次内循环比较n-1次，第二次内循环比较n-2次...最后一次内循环比较1次 =》共比较(n-1+1)*n/2=n^2/2次
	=》时间复杂度O(n^2)

4、空间复杂度
	在算法执行过程中只使用了一个变量从来存储返回的信息 =》O(1)
